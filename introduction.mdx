# Introduction to Chunker

Chunker is a powerful Java application that allows you to convert Minecraft worlds between Java Edition and Bedrock Edition. It provides both a desktop application with a simple interface and a command-line tool for automated conversions.

## What is Chunker?

Chunker enables seamless world conversion between different Minecraft editions and versions. Whether you're moving from Java to Bedrock or vice versa, Chunker handles the complex process of translating blocks, biomes, and world data between formats.

<Note>
Chunker is maintained by Hive Games and receives funding from Mojang Studios. It's trusted by the Minecraft community and documented in the official [Microsoft Creator Docs](https://learn.microsoft.com/en-us/minecraft/creator/documents/chunkeroverview?view=minecraft-bedrock-stable).
</Note>

## Key Features

- **Cross-Edition Conversion**: Convert worlds between Java Edition and Bedrock Edition
- **Version Flexibility**: Upgrade or downgrade worlds between different game versions
- **Extensive Format Support**: Supports a wide range of Minecraft versions from 1.8 to 1.21+
- **Multiple Interfaces**: Choose between a desktop app or command-line interface
- **Advanced Configuration**: Customize block mappings, world settings, pruning, and more
- **Memory Optimization**: Configurable memory usage with automatic defaults
- **Web-Based Settings**: Export and import advanced settings from https://chunker.app

## Use Cases

### Cross-Platform Play
Convert your Java Edition worlds to Bedrock Edition to play on consoles, mobile devices, or with friends on different platforms.

### World Migration
Move your builds and creations between different Minecraft versions without losing progress.

### Server Management
Automated conversion for server administrators managing multiple world formats.

### Content Creation
Map makers and content creators can reach wider audiences by supporting both editions.

## Current Limitations

<Warning>
The following features have limited or no conversion support:

- **Entities**: Most entities except paintings and item frames are not converted
- **Structure Data**: Villages, strongholds, and other structure data is not preserved

These limitations are inherent to the differences between Java and Bedrock Edition world formats.
</Warning>
